Welcome! This presentation is about HTML5 with an nod towards how HTML5 relates to mobile application development.

My Goal for this Presentation: To show you what is possible with HTML5 and to provide the HTML5 background, framework and resources for you to be able to learn and evaluate HTML5 web app development.

Caveats:

  • This is a long presentation (over 100 slides).
  • This is a "living" document. I update it with new links and examples, re-arrange the content, change my opinions and predictions (so they are always right), and otherwise fight a losing battle against bit rot. If you want a historical version then clone the sources.
  • I am not a full time iOS or Android app developer so my opinions on mobile HTML5 are just that: opinions. However, I do know a little about HTML5 and have been involved in some small ways in the development of HTML5. I have also developed a relatively popular (within its small niche) HTML5 web application (noVNC) that works on mobile devices.
  • In addition to being very long, this presentation is rendered on a single web page using jmpress and makes use of CSS3 styles to create a more interesting layout than traditional slideshows. As such, it can be stressful on your browser. I recommend using a reasonably powerful computer and a recent webkit based browser such as Chrome to get the full effect.
  • I link to many examples in this presentation. Some example will only work in recent versions of Chrome (e.g. Chrome 17). Most will work in a recent versions Firefox (e.g. Firefox 12).
